# ops-homelab-media

Docker Compose stack and supporting systemd units for a self-hosted homelab covering media (Plex, Sonarr, Radarr, Prowlarr, qBittorrent, Audiobookshelf), home automation (Home Assistant), DNS filtering (Pi-hole), and file sync (Syncthing). Includes a scheduled backup of container appdata to OneDrive via rclone.
