# app-bookbuddy-web

BookBuddy is a self-hosted companion for reading on a Kindle or with a physical book. It imports selected EPUB chapters, prepares spoiler-safe “Previously on” recaps and a short set of typed retrieval-practice questions, and offers a small spaced-review warm-up before the next reading session. The mobile-first app tracks each book’s reading position and quietly prepares the next chapter so the reader never has to manage decks or generation jobs.
